1- Problme de rgression positive

On considre le modle de rgression normale $$ y=X\theta+\sigma\xi,\;\;\xi\sim \mathcal{N}(0, I_m),$$ o $X\in \mathbb{R}^{m\times n}$ et $y\in \mathbb{R}^m$ sont les observables, et $\theta\in \mathbb{R}_+^n$ est le paramtre inconnu.

Dans les moindres carrs positifs, on modlise $\theta$ comme un vecteur valeurs positives et donc on ne cherche que les solutions positives du problme des moindres carrs. On veut donc rsoudre le problme d'optimisation: $$\tag{P1} \min_{\theta\in \mathbb{R}^n} \|X\theta-y\|_2^2,~~\;\text{sous contrainte}\;\theta\geq0.$$

def probleme_P1(X, y, n, verbose=False):
    """
    Rsout le problme (P1) de moindres carrs positifs :
    minimiser ||X @ theta - y||_2^2 sous la contrainte theta >= 0.

    Args:
        X (numpy.ndarray): matrice d'observation de taille (m, n).
        y (numpy.ndarray): vecteur d'observations de taille m.
        n (int): dimension de theta.
        verbose (bool): active la sortie verbeuse du solveur (par dfaut False).

    Returns:
        list: [status (str),
               valeur_probleme (float),
               theta_estime (numpy.ndarray),
               residuals (numpy.ndarray),
               norme_inf (float)]
    """
    # Dfinition du problme d'optimisation
    theta = cp.Variable(n)
    L = cp.norm(X @ theta - y, 2) ** 2
    objective = cp.Minimize(L)
    contrainte_positivite = theta >= 0
    prob = cp.Problem(objective, [contrainte_positivite])

    # Rsolution du problme
    prob.solve(verbose=verbose)

    # Extraction des rsultats
    status = prob.status
    valeur_probleme = prob.value if prob.value is not None else float("nan")
    theta_estime = theta.value if theta.value is not None else np.zeros(n)
    residuals = y - X @ theta_estime
    norme_inf = np.linalg.norm(theta_estime, np.inf)

    return [status, valeur_probleme, theta_estime, residuals, norme_inf]

3- Ajout de contraintes

Afin d'amliorer la stabilit du modle, on dcide d'ajouter la contrainte que la somme des valeurs de theta ne doit pas dpasser une constante $S$. le problme devient alors $$\tag{P2} \min_{\theta\in \mathbb{R}^n} \|X\theta-y\|_2^2,~~~\;\text{sous contrainte}\;\theta\geq0 \text{ et } \sum_{i=1}^n \theta_i \leq S.$$

def probleme_P2(X, y, S, n, verbose=False):
    """
    Rsout le problme d'optimisation (P2) : ||X*theta - y||^2_2,
    avec les contraintes theta >= 0 et sum(theta) <= S.

    Args:
        X (numpy.ndarray): Matrice de taille m*n.
        y (numpy.ndarray): Vecteur de taille m.
        S (int or float): Valeur contraignant la somme des coefficients de theta.
        n (int): Taille du vecteur theta.
        verbose (bool, optional): Affiche les dtails de la rsolution si True. Par dfaut False.

    Returns:
        list: Contient :
            - status (str): Statut de l'optimisation.
            - valeur_probleme (float): Valeur minimise du problme.
            - theta_estime (numpy.ndarray): Solution optimise pour theta.
            - residuals (numpy.ndarray): Rsidus y - X*theta.
            - norme_inf (float): Norme infinie de la solution estime.
    """
    # Dfinition du problme d'optimisation
    theta = cp.Variable(n)
    L = cp.norm(X @ theta - y, 2)**2
    objective = cp.Minimize(L)
    contrainte_positivite = theta >= 0
    contrainte_stabilite = sum(theta) <= S
    prob = cp.Problem(objective, [contrainte_positivite, contrainte_stabilite])
    
    # Rsolution du problme
    prob.solve(verbose=verbose)
    
    # Extraction des rsultats
    status = prob.status
    valeur_probleme = L.value
    theta_estime = theta.value
    residuals = y - X @ theta_estime
    norme_inf = np.linalg.norm(theta_estime, np.inf)

    return [status, valeur_probleme, theta_estime, residuals, norme_inf]

def probleme_P2_dual(X, y, S, n, verbose=False):
    """
    Rsout le problme d'optimisation (P2) avec retour sur les valeurs duales.

    Args:
        X (numpy.ndarray): Matrice de taille m*n.
        y (numpy.ndarray): Vecteur de taille m.
        S (int or float): Valeur contraignant la somme des coefficients de theta.
        n (int): Taille du vecteur theta.
        verbose (bool, optional): Affiche les dtails de la rsolution si True. Par dfaut False.

    Returns:
        list: Contient :
            - status (str): Statut de l'optimisation.
            - valeur_probleme (float): Valeur minimise du problme.
            - theta_estime (numpy.ndarray): Solution optimise pour theta.
            - residuals (numpy.ndarray): Rsidus y - X*theta.
            - norme_inf (float): Norme infinie de la solution estime.
            - dual_positivite (numpy.ndarray): Valeurs duales pour la contrainte de positivit.
            - dual_stabilite (float): Valeur duale pour la contrainte de stabilit.
    """
    # Dfinition du problme d'optimisation
    theta = cp.Variable(n)
    L = cp.norm(X @ theta - y, 2)**2
    objective = cp.Minimize(L)
    contrainte_positivite = theta >= 0
    contrainte_stabilite = sum(theta) <= S
    prob = cp.Problem(objective, [contrainte_positivite, contrainte_stabilite])

    # Rsolution du problme
    prob.solve(verbose=verbose)

    # Extraction des rsultats
    status = prob.status
    valeur_probleme = L.value
    theta_estime = theta.value
    residuals = y - X @ theta_estime
    norme_inf = np.linalg.norm(theta_estime, np.inf)
    dual_positivite = prob.constraints[0].dual_value
    dual_stabilite = prob.constraints[1].dual_value

    return [status, valeur_probleme, theta_estime, residuals, norme_inf, dual_positivite, dual_stabilite]
